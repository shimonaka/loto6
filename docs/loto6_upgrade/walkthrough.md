# Loto 6 Predictor Upgrade Walkthrough

## 概要
既存の「Loto 6 統計予想システム」を改修し、以下の機能を実現しました。
1.  **自動データ取得**: 公式サイトから最新データを自動スクレイピング。
2.  **Streamlitアプリ化**: 使いやすい最新のUIでの予測システム。

## 含まれるシステム

### 1. Streamlit アプリ (新機能)
**起動方法**: デスクトップの `loto6` フォルダにある **`run_app.bat`** をダブルクリック。

- **機能**:
    - **ダッシュボード**: 最新の当選数字、分析データ数を一目で確認。
    - **データ更新機能**: サイドバーの「データ更新」ボタンで、いつでも最新データを取り込めます。
    - **インタラクティブ分析**: 出現頻度や間隔のグラフをタブで切り替え可能。
    - **6つの予想アルゴリズム**: ハイブリッド、ホット、コールドなどの理論に基づいた予想が可能。

### 2. 従来型 HTML アプリ
**起動方法**: **`start.bat`** をダブルクリック。

## ファイル構成

- `loto6_app.py`: Streamlitアプリのソースコード。
- `update_loto6.py`: データ取得用Pythonスクリプト。
- `loto6_data.js`: 取得したデータの保存ファイル。
- `requirements.txt`: 必要なライブラリ一覧。

## 検証結果

- **Streamlit動作**: 起動、データ読み込み、グラフ描画、全アルゴリズムでの予想生成が正常に動作することを確認しました。
- **データ更新**: アプリ内ボタンおよびスクリプト経由で、最新データ（2025年12月時点）が反映されることを確認しました。
